# House Price Regression

## О проекте

Модель предсказания оттока стоимости квартиры.

## Цель

Предсказать стоимость квартиры.

## Стек

pandas, numpy, sklearn, matplotlib, seaborn

## Что сделано

- EDA
- ML модель

## Результат

R²: 0.9878
MAE: 0.263 млн руб

## Вывод

Определены ключевые факторы цены: площадь, район (Центральный >> Невский), тип ремонта
